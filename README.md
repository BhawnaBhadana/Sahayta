# Sahayta

**Sahayta** is a web-based platform designed to provide immediate assistance during emergencies. It connects users in need of help with resources, alerts, and local authorities in real-time. The project focuses on safety, accessibility, and community support.  

---

## Features

- **SOS Alerts:** Instantly send an SOS alert to nearby authorities or contacts.  
- **Resource Locator:** Find nearby hospitals, shelters, and essential services.  
- **Local Map Integration:** Real-time map view of resources and help points.  
- **User Authentication:** Secure login and registration system.  
- **Admin Dashboard:** Manage resources, alerts, and user requests.  

---

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript, EJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **APIs:** Google Maps API (for location services)
